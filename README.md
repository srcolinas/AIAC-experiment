# AIAC-experiment

I want to dive deeper into different ways to use AI assisted coding in my workflow as a profesional engineer. So I decided to create an application following different approaches and have my (biased) opinion on how well the coding tool helped me. Note that my thoughts will be limited to an application that was built from scratch, which is not the only type of work we developers do.

## The Application

The application is an online game inspired by Settlers of Catan, but it senters around the Tayrona people, its economy and ways of building.
